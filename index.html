⚙️ Desain Sistem Random Forest

Alur kerja sistem dibangun mulai dari input data, pengolahan, proses prediksi, hingga rekomendasi tindak lanjut.

1

Input Data

Admin atau pengguna memasukkan data akademik dan non-akademik mahasiswa ke dalam form atau upload dataset.

2

Preprocessing

Data dibersihkan, dicek kelengkapannya, lalu disiapkan agar sesuai dengan kebutuhan model prediksi.

3

Model Random Forest

Sistem memproses data melalui sejumlah decision tree untuk mengenali pola risiko dropout mahasiswa.

4

Hasil Prediksi

Model menghasilkan skor risiko dan kategori mahasiswa, yaitu risiko rendah, sedang, atau tinggi.

5

Rekomendasi

Sistem memberikan saran tindak lanjut agar kampus dapat melakukan intervensi lebih cepat dan tepat.

Catatan: versi ini adalah prototype HTML, CSS, dan JavaScript dengan logika simulasi prediksi dan dukungan upload CSV. Untuk implementasi Random Forest yang sesungguhnya, bagian prediksi dapat dihubungkan dengan backend Python seperti Flask atau Django.
